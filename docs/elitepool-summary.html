ElitePool

Unity project summary generated from repo evidence only.

Who It’s For

Players who want a mobile-friendly single-device pool game with local play, AI practice, and short challenge levels.

What It Is

ElitePool is a Unity 2022.3 pool game project with scene-based modes for PvP, PvAI, and challenge play. Repo evidence shows a 3D table/ball simulation, mobile UX helpers, and menu-driven scene flow rather than a backend-connected live service.

What It Does

  • Offers separate scenes for `Start`, `MainMenu`, `PvP_Mode`, `PvAI_Mode`, `Challenges_Mode`, and `Testing`.
  • Runs 8-ball style turn, foul, scoring, and win/loss logic through `GameStateManager`.
  • Simulates 3D ball motion, friction, spin, pockets, and rack resets via gameplay/manager scripts.
  • Includes an AI opponent with easy, medium, and hard aiming behavior.
  • Supports scripted challenge levels using `ChallengeLevelData` assets and unlock progress in `PlayerPrefs`.
  • Shows in-game UI for score, turns, challenge text, restart, win/lose, and foul feedback.
  • Adds mobile-focused helpers such as haptics, safe-area fitting, orientation warning, and performance tuning.

How It Works

Scenes/UI: `MainMenuController`, `MenuManager`, and `SceneTransitionManager` move players between menu and gameplay scenes with fade/loading behavior.

Core loop: `CueStickController3D` drives shots; `Ball3D` handles rigidbody state/collisions; `PoolGameManager3D` applies ball physics and detects end-of-turn; `GameStateManager` evaluates rules, fouls, score, and victory.

Modes: `AIPlayer` acts on Player 2 turns in PvAI; `ChallengeManager` spawns cue/target balls from `ChallengeLevelData` ScriptableObjects for challenge scenes.

State/data: runtime state is mainly in scene objects/singletons; challenge unlocks are stored in `PlayerPrefs`; backend/network service code was Not found in repo.

How To Run

  1. Open the project in Unity `2022.3.62f3`.
  2. Let Unity restore packages from `Packages/manifest.json`.
  3. Open `Assets/Scenes/Start.unity` or use the enabled scenes listed in `ProjectSettings/EditorBuildSettings.asset`.
  4. Press Play, then use the main menu to enter PvP, PvAI, or Challenges.
